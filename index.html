What I Do

Swift

객체 지향 프로그래밍과 UIKit에 대한 이해와 오류처리, 동시성 처리 그리고 메모리 관리를 Swift를 활용하여 처리할 수 있습니다.

C++

기본 문법과 개념, 함수, 포인터와 참조를 이해하며 동적으로 메모리를 관리할 수 있습니다.

JavaScript

기본 문법과 개념, 함수, 이벤트 처리를 할 수 있으며 프론트엔드 프레임워크와 라이브러리에 대한 기초적인 이해가 있습니다.

iOS

UIKit에 대한 이해와 Auto layout에 대한 처리를 할 수 있고, 네트워킹 API를 사용하는 방법과 활용법을 알며 Core Data와 같은 지속적인 데이터 저장 방법을 알고 활용할 수 있습니다.

HTML

HTML의 기본 구조를 이해하고 태그와 속성을 사용할 줄 알며 사용자 입력을 처리하기 위한 폼과 입력 요소 사용 방법을 이해하고 있습니다. 또한 기본적인 문제 해결 및 디버깅 능력을 갖추고 있습니다.

CSS

CSS 기본 문법을 알고, 선택자의 종류와 사용법을 알고 있으며, 박스 모델에 대한 이해와 코드를 체계적이고 유지관리하기 쉽게 작성하는 방법을 알고있고 실제 활용하고 있습니다.

Node.js

Node.js의 기본적인 이해와 비동기, 이벤트 기반 아키텍처, 싱글 스레드 모델에 대한 기본적 지식을 알고있고 npm을 사용하여 의존성을 관리할 수 있습니다. 또한 DB와의 연동을 할 줄 알고 코드 디버깅과 문제 해결 능력이 있습니다.

MySQL

SQL 기본 문법을 이해하고 있으며 DB 및 테이블을 생성하고 데이터를 정렬 및 필터링 할 수 있습니다. JOIN 연산을 사용하여 관련 데이터를 결합하는 방법을 알고있으며 DB 정규화에 대하여 이해하고 활용할 수 있습니다.
